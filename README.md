# job-hunt-coach — 求职教练

从 HR 和用人负责人的筛选视角出发，帮候选人成为"对方愿意面试和雇佣"的人。

定位为 career-advisor → study-planner → learning-coach 链条的最终下游——你已学完、有代码产出，缺的是求职落地。

## 什么时候用

- 学习阶段完成，准备开始找工作
- 需要简历包装（从 HR 30 秒扫描视角）
- 需要内推策略和勾搭话术
- 需要面试复盘和薪酬谈判框架
- 需要反向背调公司

## 前置条件

必须先完成 learning-coach 的学习阶段（有项目代码和 PROGRESS.md），否则它无法基于你的实际产出做求职策略。

## 安装

```bash
cp -r job-hunt-coach ~/.trae/skills/
```

## 使用

```
Use Skill: job-hunt-coach
```

覆盖全流程：搜 → 断 → 简历 → 内推 → 自我介绍 → 投递追踪 → 技术面 → HR面 → 面试复盘 → 薪酬谈判 → Offer对比 → 反向背调 → 背调准备 → 存档。按需触发，问什么走什么。

## 声明边界

| 做 | 不做 |
|----|------|
| HR视角求职策略 | 模拟面试 |
| 简历审查 + 包装话术 | 实时岗位列表（教自己搜） |
| 内推策略 + 勾搭话术 | 诊断学习方向（找 career-advisor） |
| 面试复盘 + 弱点追踪 | 设计学习路径（找 study-planner） |
| 薪酬谈判 + offer对比 | 保证结果 |

## 已验证

基于 career-advisor 案例贯穿测试：Java 35 岁空窗 1 年广州 → 完成 learning-coach → 档案刷新 → 简历包装 + 内推策略 + HR 面话术 + 薪酬谈判框架 → 完整求职闭环通过。
